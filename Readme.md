#Kreiserkennung - Bildverarbeitungs-Tool
Dieses Projekt ist weitestgehend ein Front-End für AForge.net filter und funktionen, dient aber auch dazu eigene Filter und Funktionen zu testen.
Primärziel bisher ist es Kreise in Bildern zu erkennen, man kann aber ohne Probleme jede Computer-Vision aufgabe implementieren.
